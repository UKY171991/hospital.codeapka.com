üìã Overview

The OPD API provides a comprehensive set of endpoints for managing outpatient department operations including doctors, patients, medical reports, and billing.

Base URL

https://hospital.codeapka.com/umakant/opd_api/

Response Format

All endpoints return JSON responses:

{
    "success": true,
    "data": { ... },
    "message": "Operation successful"
}

Authentication

Session-based authentication is required for write operations (POST, DELETE).

üë®‚Äç‚öïÔ∏è Doctors API

File: doctors.php

GET ?action=list

Get paginated list of doctors (DataTables compatible)

GET ?action=get&id={id}

Get single doctor details

POST ?action=save

Create or update doctor

Fields: name, qualification, specialization, hospital, contact_no, phone, email, address, registration_no, status

POST ?action=delete&id={id}

Delete doctor

POST ?action=toggle_status&id={id}

Toggle doctor active/inactive status

GET ?action=stats

Get doctor statistics (total, active, inactive, specializations, hospitals)

üí∞ Billing API

File: billing.php

GET ?action=list

Get paginated list of bills

GET ?action=get&id={id}

Get single bill details

POST ?action=save

Create or update bill

Fields: patient_name, patient_phone, patient_age, patient_gender, doctor_name, consultation_fee, medicine_charges, lab_charges, other_charges, discount, paid_amount, payment_method, bill_date, notes

Auto-calculated: total_amount, balance_amount, payment_status

POST ?action=delete&id={id}

Delete bill

GET ?action=stats

Get billing statistics (total, paid, unpaid, partial, revenue, pending)

GET ?action=get_doctors

Get list of active doctors

Payment Status

Status Description
Paid Full payment received
Partial Partial payment received
Unpaid No payment received

üíª Code Examples

JavaScript (jQuery)

// Get doctors list
$.ajax({
    url: 'opd_api/doctors.php',
    type: 'GET',
    data: { action: 'list' },
    success: function(response) {
        if (response.success) {
            console.log(response.data);
        }
    }
});

// Create new bill
$.ajax({
    url: 'opd_api/billing.php',
    type: 'POST',
    data: {
        action: 'save',
        patient_name: 'John Doe',
        consultation_fee: 500,
        paid_amount: 500,
        bill_date: '2025-11-24'
    },
    success: function(response) {
        if (response.success) {
            alert(response.message);
        }
    }
});

// Get dashboard stats
$.ajax({
    url: 'opd_api/dashboard.php',
    type: 'GET',
    data: { action: 'stats' },
    success: function(response) {
        if (response.success) {
            $('#totalDoctors').text(response.data.doctors.total);
            $('#totalRevenue').text('‚Çπ' + response.data.billing.revenue);
        }
    }
});

PHP

// Get patient history
$name = 'John Doe';
$url = "opd_api/patients.php?action=history&name=" . urlencode($name);
$response = file_get_contents($url);
$data = json_decode($response, true);

if ($data['success']) {
    $reports = $data['data']['reports'];
    $bills = $data['data']['bills'];
}

cURL

# Get doctors stats
curl "https://hospital.codeapka.com/umakant/opd_api/doctors.php?action=stats"

# Create new report
curl -X POST "https://hospital.codeapka.com/umakant/opd_api/reports.php" \
  -d "action=save" \
  -d "patient_name=Jane Doe" \
  -d "doctor_name=Dr. Smith" \
  -d "diagnosis=Common Cold" \
  -d "report_date=2025-11-24"
